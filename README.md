# 🎮 Campo Minado em Python (Pygame)

Projeto de um jogo Campo Minado desenvolvido em Python utilizando a biblioteca Pygame, com interface gráfica interativa.

## 🚀 Funcionalidades

* 🎮 Menu interativo com seleção de dificuldade
* 🧠 Primeira jogada segura (sem bomba)
* 🌊 Expansão automática de áreas vazias
* 🚩 Sistema de bandeiras (clique direito)

## 🛠️ Tecnologias utilizadas

* Python
* Pygame

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/campo-minado.git
```

2. Entre na pasta:

```bash
cd campo-minado
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Execute o jogo:

```bash
python main.py
```

## 🎯 Controles

* Clique esquerdo → Abrir casa
* Clique direito → Colocar/remover bandeira
